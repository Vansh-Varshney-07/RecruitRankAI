# RecruitRankAI

An AI-powered recruitment ranking system that combines semantic search, vector embeddings, ChromaDB, and local LLM reasoning to intelligently rank candidates beyond keyword matching.

## Features

- Resume ingestion
- Job description understanding
- Semantic search
- Hybrid candidate ranking
- Local LLM inference
- ChromaDB vector storage

## Tech Stack

- Python
- Ollama
- Qwen2.5-Coder
- nomic-embed-text
- ChromaDB
- uv
- WSL2
